# ₹ Indian Income Tax Calculator FY 2024-25

A comprehensive income tax calculator for India supporting both New and Old tax regimes.

## 🚀 Live Demo
[Dakshjain2006.github.io/tax-calculator](https://Dakshjain2006.github.io/tax-calculator)

## ✨ Features
- **New Regime** (FY 2024-25) with ₹75,000 standard deduction
- **Old Regime** with HRA, 80C, 80D, and other deductions
- Age groups: General / Senior (60-80) / Super Senior (80+)
- Section 87A rebate logic
- 4% Health & Education Cess
- Animated donut chart for effective tax rate
- Slab-wise tax breakdown table
- Monthly in-hand salary calculation

## 🛠 Tech Stack
HTML5 · CSS3 · Vanilla JavaScript · HTML5 Canvas API
